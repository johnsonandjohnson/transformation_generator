# Data Linage Generator 

`generate_data_linage.py`
usage:
```
python generate_data_linage.py [--config_path] config_path [--mapping_sheet_path] mapping_sheet_path [--output] output

Options and arguments
--config_path: Path to config directory containing a list of target tables that will be generated.
--mapping_sheet_path: Path to base directory containing data mapping sheets
--output: Base output directory for SQL files and Bash scripts
```
